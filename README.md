# Nifty 50 Stock Price Prediction using ElasticNet

## Project Overview
This repository contains a comprehensive machine learning pipeline designed to analyze and predict the next-day closing prices of Nifty 50 stocks. The project transitions from automated data ingestion to sophisticated feature engineering and model optimization, specifically targeting high-cap assets like **RELIANCE.NS**[cite: 1].

---

## Key Features

### 1. Automated Data Acquisition
*   Utilizes the `yfinance` library to fetch 5-year historical market data for top Nifty 50 tickers[cite: 1].
*   Implements automated directory management to store and organize datasets systematically[cite: 1].

### 2. Advanced Feature Engineering
To capture market psychology and price trends, the pipeline calculates several technical indicators[cite: 1]:
*   **Momentum:** Daily percentage change in closing prices[cite: 1].
*   **Trend Analysis:** 10-day and 50-day Simple Moving Averages (SMA) to smooth price noise[cite: 1].
*   **Risk Metrics:** 10-day rolling volatility (standard deviation) of daily returns[cite: 1].
*   **Price Action:** Calculations for the daily price range and the "body size" of candlesticks (Difference between Open and Close)[cite: 1].

### 3. Data Refinement & EDA
*   Performs outlier detection and removal using `seaborn` visualizations to ensure model stability[cite: 1].
*   Handles missing data generated by rolling window calculations to prepare a clean training set[cite: 1].

### 4. Machine Learning Pipeline
*   **Model:** Employs **ElasticNet Regression**, which combines the penalties of Lasso (L1) and Ridge (L2) regression[cite: 1].
*   **Optimization:** Uses `GridSearchCV` to find the optimal alpha (regularization strength) and l1_ratio[cite: 1].
*   **Normalization:** Features are standardized using `StandardScaler` to improve gradient descent convergence[cite: 1].
